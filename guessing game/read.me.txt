# 🎯 Number Guessing Game (Python)

A simple and fun command-line **Number Guessing Game** built using Python.
The program generates a random number between **1 and 100**, and the player tries to guess it with hints provided after each attempt.

---

## 🚀 Features

* 🔢 Random number generation between 1–100
* 📈 Hints for each guess (Too High / Too Low)
* 🔁 Unlimited attempts until correct guess
* ⚠️ Input validation (handles invalid inputs gracefully)
* 🎉 Displays number of attempts after winning
* 🔄 Option to play again

---

## 🛠️ Tech Stack

* Python (Standard Library)

  * `random` module

---

## 📂 How to Run

1. Make sure Python is installed on your system
2. Clone this repository:

```bash
git clone https://github.com/mrrendex2006/python-beginner-to-advance.git
cd python-beginner-to-advance
```

3. Run the script:

```bash
python guessing_game.py
```

---

## 🎮 How to Play

* The computer will think of a number between **1 and 100**
* You need to guess the number
* After each guess:

  * 📉 “Too Low” → Guess higher
  * 📈 “Too High” → Guess lower
* Keep trying until you guess correctly
* The game will show how many attempts you took
