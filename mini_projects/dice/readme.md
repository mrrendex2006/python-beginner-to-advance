🎲 Dice Rolling Game (Python)

A simple and interactive command-line dice rolling game built using Python. This project is great for beginners learning about loops, functions, user input, and control flow.

🚀 Features

🎲 Random dice roll (1–6)

🔁 Option to replay the game

❓ Confirmation system (asks user 3 times before exiting)

📊 Tracks number of attempts

💬 Interactive user messages

🧠 How It Works

User is asked if they want to roll the dice.

If "y":

A random number between 1–6 is generated.

Output message is shown based on the number.

If "n":

User is asked 3 times to confirm their decision.

If they change to "y" → dice rolls.

If still "n" → game exits with total attempts.

After each round, user can choose to continue or quit.
