# 🔢 NumPy Practice (Python Libraries)

This file contains my hands-on practice with the **NumPy** library, covering everything from basic array creation to advanced operations like reshaping, slicing, and random distributions.

---

## 🚀 Topics Covered

### 📌 1. Array Basics

* Creating arrays using lists and tuples
* Checking array type and version
* Understanding dimensions (0D, 1D, 2D, 3D)

---

### 📐 2. Array Dimensions & Shape

* `.ndim` → Number of dimensions
* `.shape` → Structure of array
* Using `ndmin` to define dimensions

---

### 🔍 3. Indexing & Accessing Elements

* Accessing elements in 1D, 2D, and 3D arrays
* Using indexing `[row, column]`

---

### ✂️ 4. Slicing Arrays

* Basic slicing
* Negative indexing
* Step slicing

---

### 🧬 5. Data Types in NumPy

* Checking datatype using `.dtype`
* Creating arrays with specific datatype
* Converting datatypes using `.astype()`

---

### 📋 6. Copy vs View

* `.copy()` → Independent copy
* `.view()` → Linked to original array
* Understanding `.base` attribute

---

### 🔄 7. Reshaping Arrays

* Convert 1D → 2D / 3D
* Using `.reshape()`
* Using `-1` for automatic dimension
* Flattening arrays

---

### 🔁 8. Iterating Arrays

* Looping through arrays
* Nested iteration for multi-dimensional arrays
* Using `np.nditer()`

---

### 🧱 9. Array Creation Methods

* `np.zeros()`
* `np.ones()`
* `np.empty()`
* `np.arange()`
* `np.linspace()`

---

### 🔃 10. Sorting & Searching

* `np.sort()`
* `np.argsort()`
* `np.lexsort()`
* `np.partition()`

---

### 🔗 11. Joining & Splitting Arrays

* `np.concatenate()`
* `np.vstack()`
* `np.hstack()`
* `np.hsplit()`

---

### 🎲 12. Random Module

* `np.random.rand()`
* `np.random.randn()`
* `np.random.randint()`
* `np.random.random_sample()`

---

## 🛠️ How to Run

1. Make sure you have NumPy installed:

```bash
pip install numpy
```

2. Run the script:

```bash
python numpy_practice.py
```
